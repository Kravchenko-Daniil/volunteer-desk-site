Сделана публичная форма заявки.
    В форме есть ФИО, email, район, описание помощи, количество нужных людей, комментарий и согласие на обработку персональных данных.
Добавлена административная панель.
    Администратор может видеть все заявки, открывать карточку заявки, принимать или отклонять её.
Реализованы статусы заявок.
    Есть статусы: новая, принятая, отклонённая, назначенная, выполненная.
Добавлена карточка заявки.
    В карточке отображаются все данные заявки, текущий статус, назначенный волонтёр и история действий.
Сделана генерация PDF.
    Заявку можно сформировать в PDF для печати и передачи волонтёру.
Добавлен экспорт в Excel.
    Администратор может выгружать отфильтрованные заявки в Excel.
Добавлены поиск и фильтры.
    Заявки можно искать по ФИО/email и фильтровать по статусу, району, волонтёру и датам.
Сделан справочник районов.
    Районы выбираются из фиксированного списка, чтобы данные были аккуратными и удобными для фильтрации.
Добавлены роли пользователей.
    В системе есть администраторы и волонтёры с разными правами доступа.
Сделан личный кабинет волонтёра.
    Волонтёр видит свои заявки, может открыть карточку и отметить помощь как выполненную.
Волонтёр может создавать заявки.
    Он может создать заявку для себя или внести заявку за другого человека.
Добавлена история действий.
    По каждой заявке видно, кто и когда её создал, принял, отклонил, назначил волонтёра или отметил выполненной.
Добавлены всплывающие уведомления.
    После действий система показывает понятные сообщения: заявка создана, принята, отклонена, выполнена и т.д.
Добавлена базовая защита от спама.
    Используется honeypot и ограничение частоты отправки заявок.
Добавлена техническая подготовка под персональные данные.
    Сайт фиксирует согласие пользователя, дату согласия, версию политики, IP и данные браузера.
Сделаны страницы политики и согласия.
    Добавлены отдельные страницы для обработки персональных данных. Юридические тексты нужно финально проверить отдельно.
Обновлён дизайн.
    Интерфейс приведён к современному минималистичному стилю: карточки, аккуратные формы, статусы, кнопки, единая визуальная система.
Улучшена адаптивность.
    Сайт адаптирован под компьютер, планшет и телефон. На мобильных списки заявок отображаются как карточки, а не как широкие таблицы.
Добавлен Docker Compose.
    Проект можно запускать через Docker вместе с PostgreSQL.
Подготовлено развёртывание.
    Добавлены настройки для production, Gunicorn, Nginx, PostgreSQL, static-файлов и переменных окружения.
Добавлены скрипты резервного копирования.
    Есть скрипты для бэкапа и восстановления PostgreSQL.
Добавлены тесты и smoke-сценарий.
    Проверяется основной путь: заявка → администратор → волонтёр → выполнение → PDF/Excel.
