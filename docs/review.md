# Ревью проекта volunteer-desk-site

**Оценка: 7.2 / 10 — уровень middle+**

---

## Сильные стороны

- Полный CRUD заявок со статусами и историей переходов.
- Ролевая модель (волонтёр / координатор / администратор).
- Экспорт в PDF и Excel.
- Docker + prod-обвязка (Nginx, Gunicorn, docker-compose.prod.yml).
- Покрытие тестами ключевых flows.
- Антиспам на формах заявок.
- Подготовка под требования ПДн (структура хранения, разграничение доступа).

- [ ] `Dockerfile` — контейнер запускается под `root`, нет директивы `USER`.
- [ ] `db.sqlite3` отсутствовал в `.gitignore` (исправлено в этой сессии, но база могла попасть в историю).
- [ ] `templates/thanks.html` — текст «Спасибо» не совпадает с формулировкой ТЗ.
- [ ] `requirements.txt` без хешей и пинов версий — нет воспроизводимой сборки.
- [ ] CI отсутствует — тесты не запускаются автоматически при пуше.
